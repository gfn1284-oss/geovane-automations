README - Deploy rápido na Vercel
1) Vá até https://vercel.com e crie uma conta (ou entre).
2) Clique em "Add New..." > "Project" > "Import".
3) Escolha "Upload" e envie este arquivo .zip.
4) Quando o Vercel pedir o diretório de saída, selecione a raiz (não mude nada).
5) Clique em "Deploy" — seu site estará online em segundos!

Se quiser mudar o link do formulário, abra o arquivo index.html e troque o link dentro da tag <a class="btn btn-primary">.
